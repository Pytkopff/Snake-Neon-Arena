Thank you for your interest in contributing! This project is a Farcaster Frame Game built on Base.

## Tech Stack
- **Framework:** Next.js
- **Blockchain:** Base Network
- **SDK:** Thirdweb
- **Database:** Supabase

## How to run locally
1. Clone the repo.
2. Install dependencies: `npm install`
3. Set up `.env` (see `.env.example`).
4. Run development server: `npm run dev`

## Pull Requests
Please create a new branch for your feature and submit a PR to `main`.
